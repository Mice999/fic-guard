# Pattern: 截图传播 / 二次创作骚扰

## 模式描述

不同于前两种模式，这种行为的核心不是"分发内容获取流量"，而是"针对特定作者制造声讨"：

1. 抓取作者作品 / 评论 / 互动截图；
2. 配上引导性的标题或解读（断章取义、扭曲语义、刻意拼接）；
3. 在大众平台发酵，吸引非圈内人参与对作者的攻击。

这种模式在涉及敏感题材的同人创作中风险最高。

## 识别信号

- 你作品中某段对话 / 描写突然在大平台被讨论；
- 出现"XX 作者居然写了 XXX"这类话题，附带的截图来自你的作品但脱离上下文；
- 评论区出现明显非圈内、非读者的攻击性账号集中出现。

## 对创作者的影响

- 这不仅是版权问题，是**人身安全问题**。
- 平台压力可能导致你的账号被锁、作品被删；
- 现实身份暴露的风险显著上升。

## 应对

**预防层**——这是这类风险下最重要的一层：

- 不在作品里留任何可关联现实身份的信息（用户名、邮箱、QQ、微博常用 ID 一律隔离）；
- 作品可见性设为仅注册用户（让传播门槛 +1）；
- 敏感章节考虑只发送给特定读者（站内私信、加密压缩包等），不放公开页面；
- 跑一遍 `fic-guard safe-publish` 检查身份隔离。

**进行时层**：

- 不要在原账号正面对线。每一次解释都在给截图加新素材；
- 不要立即删作品——删了之后你失去对原始上下文的控制，对方的截图反而成了"唯一证据"。先备份再决定；
- 联系信任的朋友帮你监控舆论走向，你自己不要刷；
- 评估是否需要短期下线该账号、迁移到更私密的渠道（仅好友可见、邀请制等）。

**事后层**：

- 如果造成了实际损害（账号封禁、现实身份外泄），保留证据，咨询专业人士；
- 复盘暴露路径：是通过截图？通过 ID 关联？通过你自己其他平台的发言？下次堵掉这条路。

## 局限

这是一个**社会问题**，工具能做的事情极其有限。这一节存在的意义是提醒你：**这类风险的对抗主要不是技术对抗，是身份隔离 + 不应战 + 必要时退场**。

如果你正在经历这类事件，并且感到压力很大，请优先照顾自己的状态——找一两个信任的人聊聊。处置可以缓一缓，你的安全感更重要。
