# 已知爬取手法与应对模式

这个目录记录**手法和模式**，不指名任何具体团体或账号。

目的是让创作者识别套路，而不是给任何团体提供曝光。如果你想 PR 一份新的 pattern，请遵守以下规则（否则会被拒绝）：

- ✅ 描述行为模式（"先在 A 站抓 → 二次包装成「资源整合」→ 在 B 站建群收费"）
- ✅ 描述识别信号（"这类群通常表现为……"）
- ✅ 描述应对方法
- ❌ 不写具体群号、用户名、域名
- ❌ 不写截图（截图里几乎一定有可识别信息）
- ❌ 不写"我怀疑是谁"
